# coming-ui

coming UI 是一个使用最新设计语言的 React UI 库

- 文档地址 - https://comingui.org
- npm 地址 - https://www.npmjs.com/package/coming

## 如何在项目中使用它

```bash
$ yarn add coming
```

or

```bash
$ npm i coming
```

```jsx
import { Button } from "coming";
const ComingDemo = () => {
  return (
    <>
      <Button>Button</Button>
    </>
  );
};
```
