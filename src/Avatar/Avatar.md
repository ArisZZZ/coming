## API

| 参数  |           说明           |   类型   | 可选值 | 默认值 |
| :---: | :----------------------: | :------: | :----: | :----: |
| size  | 标题文本(支持传入富文本) | `string` |   -    |   -    |
| shape | 标题文本(支持传入富文本) | `string` |   -    |   -    |
|  src  | 标题文本(支持传入富文本) | `string` |   -    |   -    |
